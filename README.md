# Telegram Bot Service

A lightweight and efficient Telegram bot designed to run seamlessly on cloud hosting platforms.

## Features
- **High Performance:** Optimized for low latency.
- **Hosting Friendly:** Includes keep-alive functionality to ensure 24/7 uptime.
- **Secure:** Uses environment variables for configuration.

## Prerequisites
- **Node.js:** v18.0.0 or higher.
- **Telegram Bot Token:** Obtained from [@BotFather](https://t.me/botfather).

## Environment Variables
To run this bot, you must set the following environment variable in your hosting dashboard:
- `TOKEN`: Your Telegram Bot API token.
